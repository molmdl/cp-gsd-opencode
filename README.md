## What is this

This is a fork of an older version of GSD-opencode for minimal bash-only install and more fine-tuned agent permission.

For those who don't mind to approve and just want the slash commands and agents.

Place your own opencode.json with custom permission in this directory and run update_permission.sh to create agents with updated permission with a `cp-` prefix.

Note: This is a minimal copy-only installation of and older version of gsd-opencode, lacking many features of the upstream repo.
